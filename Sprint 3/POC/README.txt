GUI Use
1. Enter the # of results you would like to see
2.  Select the category of toy from the dropdown menu
3.  Press 'Confirm Paramters'
4.  Press 'Generate' and results should appear below
5.  Press 'New Search' and repeat steps 1-4 above.  Results will appear below previous results.

Automatic Use
1. Specify input file in terminal labeled as 'input.csv' per the paramters outlined in assignment 3.4
2. Program will output 'output.csv' in the same directory as the input file was located
